# Pocket Beacon

A mod that adds a necklace which acts as a pocket beacon. It only works when in the range of regular vanilla beacon. In order to select effects in the trinket, you must first put in blocks that give you points. These points translate to charges - the amount of effect levels you can select. To select more effects, you need to put additional beacons in the necklace. Configurable.

This mod is in beta and will provide more configuration options in the future.
